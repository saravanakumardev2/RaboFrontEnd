# RaboFrontEndAssignment

This Project is mainly to filter the data based on issue count.

AngularCli 8 is being implemented in it .

Steps to run the project:
1. import the project in visual studio code .

2. ope the terminal and type `npm install`

3. then type `ng serve`

4. open the browser and type the url `http://localhost:4200/`

5.choose the file `issues.csv` and type the number as in the issuecount field.

6. we will get the result based on the issue count.
